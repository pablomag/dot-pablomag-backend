## dot.pablomag :: Backend server API and CMS

### Developed using Node.js, express & handlebars

Compile typescript files:

```bash
yarn watch
```

Compile SASS:

```bash
yarn sass:watch
```

Copy files to prod, compile and, bundle typescript (cms) files:

```bash
yarn webpack
```

Run dev server:

```bash
yarn dev
```

Run prod server:

```bash
yarn prod
```

Run tests:

```bash
yarn test
```
